# UserScripts by ThatNeoByte

A collection of UserScripts for torrent trackers and chat systems, mainly focused on Unit3D-based sites.

## Script Categories

- **DarkPeers** – Scripts specific to DarkPeers
- **The Lounge** – Chat and shoutbox enhancements
- **Unit3D** – General-purpose scripts for Unit3D-based trackers

## Credits

Some scripts are original, while others are reworked or extended versions of existing scripts.
Original authors are always credited in the script headers.
